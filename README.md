# PowerShell Automation

This repository contains PowerShell scripts used for:

- Windows administration
- System health checks
- Remote troubleshooting
- Service management
- Automation tasks
- Infrastructure support

## Current Scripts

| Script | Purpose |
|---|---|
| Check-DiskSpace.ps1 | Displays free disk space |
| Get-SystemHealth.ps1 | Basic PC health report |
| Restart-ServiceSafe.ps1 | Safely restarts services |
| Cleanup-TempFiles.ps1 | Clears temporary files |

## Goals

- Build reusable admin tools
- Improve troubleshooting speed
- Practice scripting automation
- Create a professional engineering portfolio
